# Веб-сервис «Мониторинг учебного процесса группы студентов»
Практическая часть магистерской диссертации по направлению обучения «Программная инженерия».
На основе проведенного анализа стандартов и аналогов разработана система контроля посещаемости и успеваемости студентов. Сервис предназначен для преподавателей и обеспечивает удобный доступ к данным по учебному процессу.

# Общие сведения
Целью проекта является создание удобного инструмента для отслеживания прогресса студентов преподавателем. Проект включает механизмы автоматического формирования отчетов и анализа успеваемости, позволяя повысить эффективность учебной деятельности.

# Основные возможности веб-сервиса:
Автоматическое формирование отчетности по успеваемости каждой группы.
Возможность оперативного внесения изменений в расписание занятий.
Интуитивно-понятный интерфейс редактирования данных студентов и учебных курсов.
Использование AJAX-технологий для повышения скорости взаимодействия с системой.
# Как работает сервис?
Шаги для начала работы с сервисом:
1. Регистрация пользователя:После заполнения формы регистрации заявку одобряет администратор. Затем пользователь проходит авторизацию, используя созданный логин и пароль.
2. Настройка профиля:Новый пользователь первым делом настраивает личные данные, заполняя обязательные поля учетной записи.
3. Редактор дисциплин и групп:Пользователю предлагается добавить учебные курсы и соответствующие студенческие группы. Данные подгружаются автоматически благодаря технологиям AJAX, предотвращающим дублирование сведений.
4. Создание карточек успеваемости:Преподаватель формирует связь между группами и дисциплинами, создавая шаблоны будущих таблиц успеваемости.
5. Формирование расписания:Заполняется расписание занятий по группе, на основании которого строится таблица учета посещаемости.
6. Контроль успеваемости:Пользователь выбирает необходимую группу и дисциплину, переходя непосредственно к работе с таблицами успеваемости и посещения.
# Технические требования
Рекомендуемые браузеры: Chrome, Firefox, Safari. Поддерживаются современные версии браузеров.
Серверная инфраструктура: Web-сервер с поддержкой PHP и баз данных SQL.
